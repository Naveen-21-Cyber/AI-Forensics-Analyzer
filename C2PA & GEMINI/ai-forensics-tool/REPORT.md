## 2. Technical Architecture

### 2.1 Frontend (Client-Side)
-   **Core Tech**: Vanilla HTML5, CSS3, ES6 JavaScript. No heavy frameworks (React/Vue) used to ensure lightweight performance.
-   **Modules**:
    -   `metadata.js`: Parses EXIF and C2PA provenance data.
    -   `pixel_analysis.js`: Performs Noise Variance Estimation and Error Level Analysis (ELA) simulation.
    -   `frequency.js`: Implements Fast Fourier Transform (FFT) to detect GAN-generated grid artifacts.
    -   `scoring.js`: Weighted confidence filtering engine.
    -   `ui.js`: Handles DOM manipulation and visualizations.

### 2.2 Backend (Server-Side)
-   **Framework**: Python **Flask** (Microservice architecture).
-   **Key Libraries**:
    -   `Pillow (PIL)`: Image processing and deep EXIF/Lens data extraction.
    -   `numpy`: Numerical computations for variance analysis.
    -   `fpdf2`: Professional PDF report generation.
-   **Endpoints**:
    -   `/api/analyze`: Server-side verification (ELA + Metadata).
    -   `/api/report`: Generates downloadable forensic reports.
